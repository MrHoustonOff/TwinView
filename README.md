# TwinView 
WIP README
TwinView — инструмент для визуального сравнения, анализа и контроля качества рендеров и изображений. Приложение обеспечивает синхронную навигацию, поддержку различных режимов отображения (Grid/Row/Column) и инструменты попиксельного сравнения (Slicing) в реальном времени.

## Основной функционал

### 1. Импорт и управление ассетами
Поддерживается загрузка изображений через механизм Drag-and-Drop, а также прямой импорт из буфера обмена. Система автоматически валидирует типы файлов, отсеивая неподдерживаемые форматы.

#### Методы импорта:

- Drag & Drop: Перетаскивание файлов в рабочую область.
- Clipboard: Вставка изображения комбинацией Ctrl+V.
  ![FileInput](https://github.com/user-attachments/assets/cceb9981-d961-457c-9d51-6d2bc953a757)

### 2. Управление сценой (Sidebar)
Боковая панель (Sidebar) предоставляет контроль над списком загруженных ассетов (фото).

- Visibility Toggle: Мгновенное включение/выключение видимости слоя без удаления из памяти.
- Batch Actions: Возможность очистки только неактивных слотов (Purge Inactive) или полной очистки рабочей области.
![SceneManager](https://github.com/user-attachments/assets/f603d44c-fc27-41d1-9096-6346c618df9e)

### 3. Режимы отображения (Multi Mode)
В режиме `Multi` доступно три типа компоновки вьюпорта. Лейаут рассчитывается динамически в зависимости от количества активных ассетов и доступной площади экрана.

- Grid: Автоматическая матрица (Smart Grid), стремящаяся к максимизации площади тайла (Tile Area).
- Row: Горизонтальное расположение.
- Column: Вертикальное расположение.

Реализована синхронизация камеры (Global Camera State): панорамирование и зум применяются ко всем активным слотам одновременно.
